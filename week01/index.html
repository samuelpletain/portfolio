Master Mobile UX

  • Mobile-first design means focus on content and core functionalities. Responsive design does not mean mobile-first.
  • A paradigm shift happened with the creation of smartphones.
  • Mobile-first is time consuming and costly for a full rebuild but provides the best experience across all devices, addresses constraints of mobile size/bandwidth, and is scalable if things change in the future.
  • Start small and scale up.
  • Think about reachability: most users use their thumbs, making the bottom of the screen the most accessible space to use. Users use a one-handed grip, making the bottom-right part of the screen the most accessible for them.
  • Placing the navbar at the bottom of the screen can make a great difference for the users. It can create the illusion of speed and increase user satifaction.
  • Forms can be complited to fill for user on mobile. Getting rid of labels and using placeholder and floating labels can be a good way to make things less cramped. Enable auto-completed on forms. Link anchor tags to phone numbers. Think outside the box!
  • Pop-ups/pop-overs are evil! :D Keep ads simple, sleek, and unobtrusive.
